Dual-N-Back Game

This site has an application that may, with daily training, improve a person's intelligence. This application was designed according to the protocol (Dual N-Back Task) found in a new paper in the Proceedings of the National Academy of Sciences. While the findings are only preliminary, the results are startling: It may be possible to increase almost anyone's IQ (or, more accurately, gF). Because this is such an exciting finding, we have decided to make this application and make it freely available on the web. It's our hope that the application is truly helpful for people. If it is helpful for you, please send us an e-mail.
